# Contributing

Contributions are welcome.

## What to Add

Good additions should be related to at least one of:

- mobile GUI agents;
- mobile app automation;
- multimodal mobile agents;
- personal mobile agents;
- device-cloud collaborative agents;
- on-device LLM / VLM / SLM inference;
- edge inference optimization;
- mobile agent benchmarks or datasets.

## Paper Entry Template

```markdown
### Paper Title

- **Authors**:
- **Venue / Year**:
- **Topic**:
- **Device / Platform**:
- **Modalities**:
- **Agent relevance**:
- **Inference relevance**:
- **Links**:
- **Notes**:
```

## Style

- Prefer structured notes over long summaries.
- Mark uncertain metadata as `TODO` instead of guessing.
- Include paper, code, project, dataset, or benchmark links when available.
- Keep entries focused on why the work matters for mobile agents or edge inference.
