# Reading Map

## Stage 1: Understand Mobile GUI Agents

Start with:

1. AutoDroid
2. AutoDroid-V2
3. LLM-Powered GUI Agents in Phone Automation
4. AgentCPM-GUI

Key questions:

- How does the agent observe the mobile UI?
- What is the action space?
- Does the system use screenshots, UI hierarchy, or both?
- How is task success evaluated?

## Stage 2: Study Practical Deployment and Safety

Read:

1. Advancing Mobile GUI Agents
2. VeriSafe Agent

Key questions:

- How are unsafe or invalid actions detected?
- What can be verified before execution?
- What is the tradeoff between autonomy and safety?

## Stage 3: Move to On-device and Edge Inference

Read:

1. Ferret-UI Lite
2. On-Device Language Models
3. Mobile Edge Intelligence for LLMs
4. Intelligence per Watt
5. Small Language Models are the Future of Agentic AI

Key questions:

- What model size is practical on mobile devices?
- What are the main bottlenecks: compute, memory, energy, latency, or I/O?
- Which capabilities can stay local, and which should be offloaded?

## Stage 4: Study Collaborative and Device-cloud Agents

Read:

1. EcoAgent
2. Mobile-Agent-v2

Key questions:

- What should run on device?
- What should run in the cloud?
- How do multiple agents communicate?
- How does collaboration affect cost, latency, and privacy?
